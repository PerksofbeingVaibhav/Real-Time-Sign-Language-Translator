Chironomy.

A sign language interpreter is a person trained in translating between a spoken and signed language. This usually means someone who interprets what is being said and signs it for someone who can't hear, but understands the sign. In this project the user has to train gestures for initiating and terminating the conversation between the people who are not able to hear.

HOW IT WORKS?

In this project a pop-up window will be opened to your interface in which the system requires your permission in order to grant access to your camera and microphone. After allowing, the user gets directed to the 2nd page in which the system asks the user to train their gestures by making symbols in-front of the camera and clicking on the Start Gesture till it reaches 30 times and then clicking on Stop Gesture. (NOTE: - the user has to cover the whole camera window in order to make the things clear for them) Then the user is directed to another page in which he/she has to add their gesture name and thus while applying these gestures a voice command will try to tell the admin/operator which gesture they are applying.

Prequisite Knowledge

You should also have a basic knowledge about tensorflow and bootswrap framework
